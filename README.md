# Maître D' 🦉

Surprise blind restaurant dates in San Francisco, curated by an AI butler.

**[maitred.fun](https://maitred.fun)**

## What is this?

Every two weeks, Alfred — an AI concierge — scouts SF's best restaurants, books a table, and sends a calendar invite with nothing but a pickup time. No restaurant name. No cuisine hint. You find out when you arrive.

## Stack

- Static HTML + CSS (Instrument Serif + Outfit)
- Vercel serverless function for waitlist signups (`/api/signup`)
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npx vercel dev
```

Open [localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` auto-deploy to production via Vercel.
